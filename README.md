# Expense Tracker Desktop Application

A desktop application built using Python, PyQt6, and SQLite.

This project was developed while exploring Python GUI development and database integration using SQLite.

The application allows users to:
- Add expenses
- View expenses
- Delete expenses

Expense data is stored permanently using an SQLite database

## Features

- GUI-based desktop application
- Expense management system
- SQLite database integration
- Dynamic table updates
- Input validation
- Custom UI styling

## Technologies Used

- Python
- PyQt6
- SQLite
- Qt Style Sheets (QSS)

## Project Structure

main.py         # Run the application ;
app.py          # UI logic and application features ;
database.py     # Database operations ;
expense.db      # SQLite database
